# 🛡️ Cybersecurity Interview Prep & Knowledge Base
### Master the Technical Interview across Offensive, Defensive, and AppSec Domains

This repository is a curated collection of interview questions, technical deep-dives, and cheat sheets designed to help cybersecurity professionals and aspirants ace their technical interviews. It covers everything from entry-level SOC roles to advanced VAPT and AppSec positions.

## 🎯 Key Topics Covered

### 🔍 Offensive Security & VAPT
- Networking (TCP/IP, OSI Model, Common Ports)
- Web Application Attacks (XSS, SQLi, CSRF, IDOR)
- Windows & Linux Privilege Escalation
- Active Directory Attacks (Kerberoasting, Pass-the-Hash)

### 🛡️ Defensive Security & SOC
- The Diamond Model & Cyber Kill Chain
- SIEM (Splunk/ELK) Fundamentals
- Interpreting PCAP and Log files
- Malware Analysis & Sandboxing basics

### 💻 Application Security (AppSec)
- Breaking down the OWASP Top 10 (2021)
- Threat Modeling (STRIDE/PASTA)
- Security Headers & JWT Security
- SAST vs. DAST vs. IAST
